**Multi-agent orchestration for Claude Code. Zero learning curve.**

Oh My ClaudeCode (OMC) is a multi-agent orchestration layer that runs on top of Claude Code.
With a single prompt, it coordinates multiple specialized agents to automate complex development tasks.

## Quick Start

```bash
# 1. Install
/plugin marketplace add https://github.com/Yeachan-Heo/oh-my-claudecode
/plugin install oh-my-claudecode

# 2. Configure
setup omc

# 3. Run
autopilot build me a REST API for managing tasks
```

That's it. The rest is automatic.

## Key Features

| Feature | Description |
|---------|-------------|
| **19 Specialized Agents** | Role-based division: explore, plan, implement, verify, review |
| **28 Skills** | Automated workflows like autopilot, ralph, ultrawork |
| **Magic Keywords** | Run the full pipeline with a single phrase like "autopilot build me X" |
| **Team Orchestration** | Coordinate multiple Claude/Codex/Gemini agents simultaneously |
| **State Management** | Maintain context across sessions with notepad and project-memory |
| **Multi-AI** | Use Claude, Codex, and Gemini for cross-validation (optional) |

## Team Mode

The core orchestration mode of OMC. Multiple agents collaborate through a pipeline.

```bash
/team 3:executor "fix all TypeScript errors"
```

| Command | Workers | Use Case |
|---------|---------|----------|
| `omc team N:claude "..."` | Claude CLI | General tasks |
| `omc team N:codex "..."` | Codex CLI | Code review, security analysis |
| `omc team N:gemini "..."` | Gemini CLI | UI/UX, docs, large context |
| `/ccg` | Claude + Codex + Gemini | Tri-model synthesis |
