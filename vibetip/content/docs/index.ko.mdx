**Multi-agent orchestration for Claude Code. Zero learning curve.**

Oh My ClaudeCode(OMC)는 Claude Code 위에서 돌아가는 멀티 에이전트 오케스트레이션 레이어입니다.
프롬프트 하나로 여러 전문 에이전트를 조율해서 복잡한 개발 작업을 자동화합니다.

## Quick Start

```bash
# 1. 설치
/plugin marketplace add https://github.com/Yeachan-Heo/oh-my-claudecode
/plugin install oh-my-claudecode

# 2. 설정
setup omc

# 3. 실행
autopilot build me a REST API for managing tasks
```

그게 전부입니다. 나머지는 자동입니다.

## 주요 기능

| 기능 | 설명 |
|------|------|
| **19개 전문 에이전트** | 탐색, 계획, 구현, 검증, 리뷰까지 역할 분담 |
| **28개 스킬** | autopilot, ralph, ultrawork 등 자동화 워크플로우 |
| **매직 키워드** | "autopilot build me X" 한 마디로 전체 파이프라인 실행 |
| **팀 오케스트레이션** | 여러 Claude/Codex/Gemini 에이전트를 동시에 조율 |
| **상태 관리** | notepad, project-memory로 컨텍스트 유지 |
| **멀티 AI** | Claude, Codex, Gemini를 교차 검증에 활용 (선택) |

## 팀 모드

OMC의 핵심 오케스트레이션 모드입니다. 여러 에이전트가 파이프라인으로 협업합니다.

```bash
/team 3:executor "fix all TypeScript errors"
```

| 명령 | 워커 | 용도 |
|------|------|------|
| `omc team N:claude "..."` | Claude CLI | 일반 작업 |
| `omc team N:codex "..."` | Codex CLI | 코드 리뷰, 보안 분석 |
| `omc team N:gemini "..."` | Gemini CLI | UI/UX, 문서, 대용량 컨텍스트 |
| `/ccg` | Claude + Codex + Gemini | 트리-모델 합성 |
