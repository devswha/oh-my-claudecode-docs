## 스킬이란?

스킬은 `/oh-my-claudecode:스킬이름` 형식으로 호출하는 명령입니다.
매직 키워드로도 활성화되니까, 명령어를 외우지 않아도 대화 중에 바로 쓸 수 있습니다.

## 스킬 호출 방법

### 슬래시 명령

```bash
/oh-my-claudecode:autopilot "REST API for a bookstore"
/oh-my-claudecode:ralph "fix all TypeScript errors"
/oh-my-claudecode:team 3:executor "implement user auth"
```

### 매직 키워드

대화 중에 키워드를 쓰면 자동으로 활성화됩니다.

```
autopilot build me a todo app with React
ralph fix the authentication module
ulw implement user management
```

## 스킬 관계도

```
deep-interview (요구사항 정리)
  → ralplan (합의 계획)
    → autopilot (자율 실행)
      → ralph (반복 실행)
        → ultrawork (병렬 실행)
      → ultraqa (QA 사이클링)
```

상위 스킬은 하위 스킬을 자동으로 포함합니다.
autopilot을 실행하면 내부에서 ralph와 ultrawork가 함께 돕니다.
